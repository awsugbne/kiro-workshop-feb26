### Model Credit Multipliers

| Model | Multiplier | Best For |
|-------|------------|----------|
| Qwen3 Coder Next | 0.05x | Long CLI sessions, boilerplate |
| MiniMax 2.1 | 0.15x | Multilingual code, UI generation |
| DeepSeek v3.2 | 0.25x | Agentic workflows, debugging |
